# ATS Resume Tracking System

A Flask-based Applicant Tracking System (ATS) that analyzes resumes and matches them with job descriptions using NLP techniques.

---

## Features
- Resume analysis and ATS scoring
- Keyword matching between resumes and job descriptions
- Job description comparison
- Simple and user-friendly interface
- Fast and efficient processing

---

## Tech Stack
- Python
- Flask
- NLP
- HTML
- CSS

## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/Akhila-Palagani24/genai.git
```

### 2. Navigate to Project Directory
```bash
cd genai
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the Application
```bash
python app.py
```

---

## Usage
1. Upload a resume
2. Enter the job description
3. View ATS score and matching results
